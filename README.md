# ait-onboarding-app-cd
CD configuration for app from the ait-onboarding-app repo

## Usage

### Helm

Install the helm chart
```
make h-install
```

Upgrade the helm chart
```
make h-upgrade
```

Uninstall the helm chart
```
make h-uninstall
```


### ArgoCD

Install the helm chart
```
make a-install
```
